# JrubyZemberek

This is a gem for making it easy to call zemberek 2.0 nlp library functions from JRuby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jruby_zemberek'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jruby_zemberek
